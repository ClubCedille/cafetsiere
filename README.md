cafetsiere
==========

## Installation

1. Install Dependencies
    - Node.JS >= 0.10.x
    - Bower ( `npm install -g bower` )
    - Grunt ( `npm install -g grunt-cli` )
2. Run `npm install`
3. Run `bower install`
4. Copy config-example.js as config.js, changing values as needed
5. Run `grunt`

Currently listens on port 3000 (can be modified in Gruntfile)
